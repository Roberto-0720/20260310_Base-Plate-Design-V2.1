# Base Plate Design Tool

Desktop application for steel base plate design and checking per AISC 360 / ACI 318M.
Integrates directly with **SAP2000** to import joint coordinates and reaction forces.

## Architecture: Mixin Pattern

All 85 methods reference `self.xxx` extensively. Splitting into standalone
functions would require changing every `self.xxx` в†’ `app.xxx`, which
violates the "no logic changes" rule.

**Solution:** every split file defines a mixin class. `BasePlateApp` in
`app.py` inherits from all 15 mixins, so every `self.xxx` reference
continues to work without modification.

```python
class BasePlateApp(
    MainWindowMixin, TabBasePlateMixin, MaterialTabMixin,
    ColumnBaseTabMixin, WidgetsMixin, DialogsMixin,      # UI
    PlotManagerMixin, NodeManagerMixin, CalculationsMixin,
    DesignCheckMixin, HingeXlsxMixin,                    # Logic
    MaterialDataMixin, FileManagerMixin,
    Sap2000Mixin, ExcelExportMixin,                      # Data
):
    ...
```

## Workflow

1. **Column Base** tab в†’ connect to SAP2000 в†’ Load (Auto) to extract coordinates and reactions
2. **Material Definition** tab в†’ review/edit material tables from `Data.xlsx`
3. **Base Plate Detail** tab в†’ click a node в†’ assign materials and bolt size в†’ Apply to Node
4. Click **RUN DESIGN CHECK** в†’ nodes colour green (OK) or red (NG)
5. **File в†’ Export to Excel** в†’ generates `Calculation Report.xlsx` in the project folder

| File | Location | Description |

| `Fixed Type.xlsx` | same folder | Same as above for fixed-base nodes |
| `Template.xlsx` | same folder or script dir | Calculation report template |
| `Calculation Report.xlsx` | same folder | Generated by Export to Excel |
| `bpl_coordinate.csv` | same folder | Joint coordinates from SAP2000 |
| `reaction_data_sap2000.csv` | same folder | Joint reactions from SAP2000 |
| `Bpl.json` | user-chosen | Saved project file |
